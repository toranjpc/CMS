# فایل‌های استاتیک

این پوشه برای فایل‌های استاتیک مانند CSS، JS، تصاویر و ... استفاده می‌شود.

## ساختار پیشنهادی

```
public/
├── css/
│   ├── bootstrap.min.css          # فایل اصلی بوت‌استرپ
│   ├── bootstrap.rtl.min.css      # نسخه RTL بوت‌استرپ (اگر نیاز دارید)
│   └── custom.css                 # فایل‌های CSS سفارشی
├── js/
│   ├── bootstrap.bundle.min.js  # فایل اصلی JS بوت‌استرپ
│   └── custom.js                 # فایل‌های JS سفارشی
└── images/                        # تصاویر (اختیاری)
```

## نحوه استفاده

بعد از قرار دادن فایل‌ها، در `nuxt.config.ts` لینک‌ها را اضافه کنید.
